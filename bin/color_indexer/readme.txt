/*
* Small C program for getting image most popular collor palete items in image
*
* Dependency 
* It depends on opencv library
* http://opencv.willowgarage.com/wiki/
*/

/**************************************/
How to compile?

cd bin/color_indexer/
./compile.sh


/**************************************/
How to use it?

Usage: color_indexer <image-file-name> <pallete-file-name> <threshold> <inside_height> <inside_width> <method>

/**************************************/

Example 1.
./bin/color_indexer/color_indexer bin/color_indexer/forest.jpg doc/color_palletes/color_pallete.txt 25 100 100 rectangle
Method can be - rectangle, or ellipse.

1. Left variable represents pallete_id
2. Right item represents how many times color item was matched
